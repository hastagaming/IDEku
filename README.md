# IDEku 🚀
**Mobile Integrated Development Environment untuk Android**

IDEku adalah lingkungan pengembangan terintegrasi (IDE) yang dirancang khusus untuk perangkat Android. Proyek ini menggabungkan kekuatan terminal emulator dengan antarmuka yang ramah pengguna untuk memudahkan koding langsung dari genggaman.

### ✨ Fitur Utama
* **Terminal Terintegrasi:** Berbasis inti terminal Termux yang stabil.
* **Optimasi Mobile:** Ringan dan responsif untuk perangkat Android (seperti Realme Note 50).
* **Rig Build Otomatis:** Otomatisasi rilis APK menggunakan GitHub Actions.
* **Dukungan I++:** Dirancang untuk mendukung pengembangan bahasa pemrograman I++.
