# IDEku 🚀
**Mobile Integrated Development Environment for Android**

IDEku is an Integrated Development Environment (IDE) specifically designed for Android devices. This project combines the power of a terminal emulator with a user-friendly interface to make coding on the go easier than ever.

### ✨ Key Features
* **Integrated Terminal:** Powered by the stable Termux terminal core.
* **Mobile Optimized:** High-performance and responsive for Android devices.
* **Auto-Build Rig:** Automated APK signing and releases via GitHub Actions.
* **I++ Support:** Designed to support the I++ programming language.
